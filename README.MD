## callmeup is a small project to demonstrate the concepts of WebRTC by facilitating video calls across the web!

## 🧐 Feature list

- [x] Video Call
- [x] Screen Sharing
- [ ] Select camera
- [ ] Select audio device

## 🚀 Getting started

This project is deployed at [callmeup.vermagaurav.co](https://callmeup.vermagaurav.co)

Guide for local deployment -

1. Clone the repository

```
git clone https://github.com/vermagaurav8/callmeup.git
```

2. Change the working directory

```bash
cd callmeup
```

3. Install http-server

```bash
npm install --global http-server
```

4. Run the app

```bash
http-sever
```


5. Open project at [localhost:8080](http://localhost:8080)

🌟 You are all set!
